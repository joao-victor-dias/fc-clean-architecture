# FC-CLEAN-ARCHITECTURE

## Descrição

Este projeto faz parte dos desafios do curso Full Cycle com foco na aplicação prática de **Clean Architecture**.

Atualmente o projeto possui:

### Use Cases de Product

Implementação de casos de uso seguindo os princípios da Clean Architecture para as entidades do sistema.

### API REST

Endpoints para:

- Customer
- Product

### Testes

- Testes unitários
- Testes de integração
- Testes E2E

### Documentação

- Swagger UI

---

## Tecnologias utilizadas

- TypeScript
- Node.js
- Express
- Jest
- Supertest
- Sequelize
- Sequelize Typescript
- SQLite (in-memory)
- Swagger

---

## Como executar o projeto

### Instalar dependências

```bash
npm install
```

### Executar testes

```bash
npm test
```

### Iniciar a API

```bash
npm run dev
```

#### A aplicação será iniciada em:

```bash
http://localhost:3000
```

#### Documentação Swagger:

```bash
http://localhost:3000/docs
```
